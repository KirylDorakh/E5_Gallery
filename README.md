# Simple Gallery

A lightweight, responsive image gallery built with vanilla HTML, CSS, and JavaScript.

---

## Features

- Image preview with next / previous navigation
- Thumbnail gallery
- Keyboard navigation (← / →)

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/KirylDorakh/E5_Gallery.git
   ```

2. 	Open index.html in your browser
            (no build tools or dependencies required)
